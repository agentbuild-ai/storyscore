## Overview

StoryScore is a communication coaching tool that scores spoken or written content across structured dimensions and provides actionable coaching feedback.

## User Flow

1. User pastes or types their content (speech, pitch, presentation script, etc.)
2. User selects a context (e.g., "Pitching to executives")
3. System runs Pass 1: analytical scoring across dimensions
4. System runs Pass 2: coaching generation based on scores
5. User sees scores + coaching feedback
